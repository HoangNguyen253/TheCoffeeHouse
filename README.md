# Ứng dụng The Coffee House (clone)
Ứng dụng được xây dựng dựa trên ứng dụng [The coffee house](https://play.google.com/store/apps/details?id=com.thecoffeehouse.guestapp) trên Google Play, chỉ được xây dựng hoàn thiện trên Android.
### Các chức năng chính:
- Đăng nhập, có xác thực OTP thông qua email, lưu OTP trong database.
- Xem, lọc đồ ăn, thức uống, khuyến mãi.
- Đặt đồ ăn/thức uống (tích hợp thông tin tỉnh thành để generate địa chỉ).
- Thêm góp ý về ứng dụng.
- Quản lý thông tin bản thân.
- Xem thông tin các của hành The Coffee House.
- Quản lý các đơn hàng đã đặt.
### Công cụ sử dụng
- Visual Studio 2019
- XAMPP
### Công nghệ sử dụng
- Front-end: Xamarin Forms
- Back-end: .Net Web API
- Database: MySQL (sử dụng phpMyAdmin của XAMPP lưu thông tin tỉnh thành) và SQL Server (Lưu thông tin ứng dụng).
